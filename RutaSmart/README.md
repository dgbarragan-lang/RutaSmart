# RutaSmart Landing Page

- **Estudiante:** Daniel Barragan
- **Tema asignado:** Operadora logística inteligente para e-commerce
- **Descripción:** Landing page profesional para RutaSmart, una empresa ficticia que ofrece servicios logísticos completos para tiendas en línea. La página presenta los servicios, beneficios, planes de cobertura, testimonios y un formulario de contacto para solicitar cotización.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 externo (modularizado en `variables.css`, `layout.css` y `componentes.css`)
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)

## Estructura del proyecto

- `index.html`
- `css/`
  - `variables.css`
  - `layout.css`
  - `componentes.css`
- `img/`
- `README.md`

## Instrucciones para abrir la página

1. Abrir la carpeta del proyecto en el explorador de archivos.
2. Hacer doble clic en `index.html` para abrirlo en el navegador.
3. Alternativamente, abrir la carpeta en VS Code y usar una extensión como Live Server para vista en vivo.

## Observaciones de cumplimiento

### Requisitos cumplidos

- La página utiliza una estructura semántica correcta con `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` y `<footer>`.
- Contiene un solo título principal `<h1>`.
- Incluye todas las secciones obligatorias:
  - Encabezado con logotipo, navegación interna y diseño responsive.
  - Hero con título principal, propuesta de valor, descripción, botón CTA e imagen.
  - Presentación de la empresa.
  - Servicios con al menos tres tarjetas, iconos, títulos, descripciones e imágenes.
  - Sección visual destacada con carrusel de planes/cobertura.
  - Indicadores destacados.
  - Testimonios con dos comentarios reales.
  - Llamada a la acción con formulario de contacto.
  - Pie de página con datos de contacto, horario, redes sociales y derechos reservados.
- Utiliza Bootstrap 5 para el diseño de contenedores, filas, columnas, clases responsive, espaciado, botones e imágenes fluidas.
- El CSS está separado en tres archivos según el requisito.
- La paleta de colores está centralizada en variables CSS en `variables.css`.
- Se utilizan iconos de Bootstrap Icons y hay más de seis iconos en la página.
- Las imágenes están en la carpeta `img/` y tienen nombres descriptivos.
- El formulario usa `label`, `input`, `textarea` y `select` correctamente.
